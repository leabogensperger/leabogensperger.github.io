Research

I'm interested in inverse problems in medical imaging as well as deep learning and generative modeling. * indicates equal contribution.

Synthetic Augmentation for Anatomical Landmark Localization using DDPMs
Arnela Hadzic, Lea Bogensperger, Simon Johannes Joham, Martin Urschler
Simulation and Synthesis in Medical Imaging (SASHIMI), 2024

A DDPM-based framework for generating medical images with corresponding landmark heatmaps to improve anatomical landmark localization. The approach introduces a novel quality assessment technique using a Markov Random Field for landmark matching and a Statistical Shape Model for landmark plausibility checks.

FlowSDF: Flow Matching for Medical Image Segmentation Using Distance Transforms
Lea Bogensperger*, Dominik Narnhofer*, Alexander Falk, Konrad Schindler, Thomas Pock
under review, 2024
arXiv

A flow matching framework for generative segmentation of medical images using signed distance functions. This integration allows the noise-injection perturbation process to canonically smooth the distribution of SDF masks.

Score-Based Generative Models for Medical Image Segmentation using Signed Distance Functions
Lea Bogensperger, Dominik Narnhofer, Filip Ilic, Thomas Pock
DAGM German Conference on Pattern Recognition, 2023   (Honorable Mention)
arXiv paper code

A score-based generative model for binary medical image segmentation using signed distance functions. Thereby, the corruptive mechanism by the diffusion model acts on SDF masks instead of the binary masks which yields a more natural distortion.

Learned Discretization Schemes for the Second-Order Total Generalized Variation
Lea Bogensperger, Antonin Chambolle, Alexander Effland, Thomas Pock
International Conference on Scale Space and Variationa Methods in Computer Vision, 2023
arXiv paper code

We introduce a framework to incorporate general discretization schemes for the second-order total generalized variation (TGV), for which we show variational consistency. Interpolation filters are then learned using a piggyback algorithm.

A joint alignment and reconstruction algorithm for electron tomography to visualize in-depth cell-to-cell interactions
Lea Bogensperger, Erich Kobler, Dominique Pernitsch, Petra Kotzbeck, Thomas R. Pieber, Thomas Pock, Dagmar Kolb
Histochemistry and Cell Biology, 2022
paper code

We develop a joint alignment and reconstruction algorithm for electron tomography that does not require the use of fiducial markers. We show its application within type 1 diabetes research to non-obese diabetic (NOD) mice to study the interaction between immune cells and pancreatic beta cells.

Convergence of a Piggyback-style method for the differentiation of solutions of standard saddle-point problems

paper preprint

On the Influence of Beta Cell Granule Counting for Classification in Type 1 Diabetes
Lea Bogensperger, Marc Masana, Filip Ilic, Dagmar Kolb, Thomas R. Pieber, Thomas Pock
Computer Vision and Pattern Analysis Across Domains, 2021
paper

We use a deep learning approach to investigate the influence of insulin granules in pancreatic beta cells of NOD mice. In a multi-task learning setting, the regression task can aid in discriminating healthy from diabetic samples.

A study on robust feature representations for grain density estimates in austenitic steel
Filip Ilic, Marc Masana, Lea Bogensperger, Harald Ganster, Thomas Pock
Computer Vision and Pattern Analysis Across Domains, 2021
paper code

We show that deep neural networks can be used to estimate grain density of austenitic steel. Thereby we compare classification and regression models, showing that the learned feature representations are vastly different.
